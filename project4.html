Extract text from image.

Problem Statement

Imagine that Paperless Logistics Inc. is a fast-growing logistics company that manages a vast network of suppliers, contractors, and clients. With the rapid expansion of their services, they have found themselves drowning in paperwork-freight bills, invoices, delivery orders, and compliance documents, to name a few.
Efficiency is taking a hit; manual data entry errors are leading to invoice disputes, and crucial contract renewals are being missed. To reclaim their competitive edge, the company has decided to digitize and automate its document processing pipeline. Their focus is on scanning and extracting key information from various kinds of documents, which they aim to achieve using AWS Textract.
My mission is to build a solution that automates the extraction of information from different types of logistics documents. This will allow Paperless Logistics Inc. to reduce manual errors, improve efficiency, and streamline their operations.

Introduction

AWS Textract is a machine learning service from Amazon Web Services (AWS) that automatically extracts textual content, forms, and tables from scanned documents. It can recognize a wide range of document formats, including PDFs and JPEGs, among others. Textract goes beyond traditional optical character recognition (OCR) to not only identify the text in the document but also understand its structure, such as fields in forms or data in tables.

Solution Architecture

Data extraction in AWS Cloud using AWS Textract and Python in AWS Cloud9 IDE. The project involved extracting data from a file stored in an AWS S3 bucket using text extraction in Python and then storing it in a DynamoDB table.

Code part

main.py

import boto3
	# Initialize the AWS services clients
	s3 = boto3.client('s3')
	textract = boto3.client('textract')
	dynamodb = boto3.resource('dynamodb')
	
	# Specify your S3 bucket and file name
	bucket_name = "data-extract-paperless-logistics"
	file_name = "commercial-invoice-template test sample.png"
	
	# DynamoDB table to store extracted text
	table_name = "data-extract-paperless-logistics-data"
	table = dynamodb.Table(table_name)
	
	def extract_text_from_s3_document():
		# Using AWS Textract to extract text from the document stored in S3
		response = textract.detect_document_text(
			Document={
				'S3Object': {
					'Bucket': bucket_name,
					'Name': file_name
				}
			}
		)
	
		# Extracted text
		extracted_text = ""
	
		# Iterate over the detected items to collect extracted text
		for item in response["Blocks"]:
			if item["BlockType"] == "LINE":
				print(item["Text"] + "\n")
				extracted_text += item["Text"] + "\n"
	
		# Storing extracted text in DynamoDB
		table.put_item(
			Item={
				'document_name': file_name,
				'extracted_text': extracted_text
			}
		)
	
		print(f"Successfully extracted and stored text for {file_name}")
	
	# Execute the function
	if __name__ == "__main__":
		extract_text_from_s3_document()
